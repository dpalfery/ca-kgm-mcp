# 1-Presentation Layer

## Purpose
Rules and standards for the presentation/UI layer of applications.

## What Goes Here
- UI component standards
- Styling guidelines
- Accessibility requirements
- Frontend framework patterns
- User interface best practices
- Visual design rules

## Topics Covered
- React/Vue/Angular components
- HTML/CSS standards
- Responsive design
- Form validation (UI)
- Button and control standards
- Layout patterns

## When to Apply
Use these rules when:
- Building UI components
- Styling pages or screens
- Implementing user interactions
- Working on frontend code
- Designing user interfaces
